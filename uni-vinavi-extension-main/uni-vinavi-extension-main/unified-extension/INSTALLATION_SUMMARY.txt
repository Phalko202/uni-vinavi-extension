## Troubleshooting

### Extension won't load
- Check all files are in the unified-extension folder
- Verify manifest.json is valid JSON
- Check Chrome Developer mode is enabled

### Module won't open
- Check browser console for errors (F12)
- Verify you're on the correct website
- Try reloading the extension

### Icons missing
- Icons are optional
- Extension will work with default Chrome icon
- Copy icons from vinavi-universal-extension/lab/icons if needed
