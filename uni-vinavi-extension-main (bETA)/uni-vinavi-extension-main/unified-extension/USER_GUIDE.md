## Troubleshooting Quick Fixes

### Module Won't Open
```
Problem: Clicked card but nothing happens
Fix: Check browser console (F12)
     Reload extension in chrome://extensions/
```

### Extract Button Disabled
```
Problem: Can't click "Extract Data"
Fix: Ensure you're on dharaka.hmh.mv
     Reload the page
     Check data is visible on page
```

### QuickText Not Expanding
```
Problem: Typed shortcut but no expansion
Fix: Verify you're on vinavi.aasandha.mv
     Reload page
     Check template is saved
```

### Lab Ordering Can't Find Patient
```
Problem: Search returns no results
Fix: Verify you're logged into Vinavi
     Check patient exists in system
     Try different search criteria
```
